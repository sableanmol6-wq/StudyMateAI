# StudyMate AI

StudyMate AI is an AI-powered study assistant built using Python, Streamlit, and Google Gemini API.

## Features

- AI Question Answering
- Mistake Tracking
- Personalized Study Planner
- AI Quiz Generator
- AI Flashcard Generator
- Progress Analytics Chart
- PDF Study Report Export

## Technologies Used

- Python
- Streamlit
- Google Gemini API
- Pandas
- Matplotlib
- ReportLab

## Purpose

This project helps students learn efficiently by providing AI-powered explanations, quizzes, flashcards, and personalized study plans.
